================================================================================
                          TROUBLESHOOTING
================================================================================

Problem: "RESEND_API_KEY not configured"
Solution: Add it to Netlify environment variables (see setup above)

Problem: Only some emails received
Solution: This is normal (resilient design). Check Resend dashboard.

Problem: Emails not received at all
Solution: 
1. Check spam folder
2. Verify email addresses in submission
3. Check Resend dashboard (https://resend.com)
4. Verify RESEND_API_KEY starts with "re_"

Problem: Emails in preview environment
Solution: Expected to fail (no API key in v0 preview). Works after Netlify deploy.
