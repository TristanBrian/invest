STEP 8: IF EMAILS DON'T ARRIVE
================================================================================
Check these in order:

1. Console logs show "API Key exists: false"?
   - RESEND_API_KEY not in Netlify environment
   - Add it to Site Settings → Build & Deploy → Environment

2. Console shows "API error: 401"?
   - API key is invalid or expired
   - Get new one from https://resend.com → API Keys
   - Update in Netlify environment

3. Console shows "API error: 500"?
   - Check Resend dashboard for account limits
   - Check if account is verified
   - Try again in 5 minutes

4. Emails go to spam?
   - Normal for first deliveries
   - Mark as "Not Spam" to improve delivery
   - Resend domain reputation will improve over time

5. No console output at all?
   - Check browser extensions blocking content
   - Try incognito mode
   - Clear browser cache

================================================================================
TROUBLESHOOTING QUICK REFERENCE
================================================================================

SYMPTOM: Form shows "success" but no emails arrive
CAUSE: RESEND_API_KEY not in environment
FIX: Add to Netlify Site Settings → Build & Deploy → Environment

SYMPTOM: Console shows "API Key exists: false"
CAUSE: Environment variable not loaded
FIX: Trigger new deploy after adding env var

SYMPTOM: Console shows "401 Unauthorized"
CAUSE: Invalid API key
FIX: Get new key from Resend, update in Netlify

SYMPTOM: Invoice fails with 410 error
CAUSE: Calling old endpoint
FIX: Code already fixed, redeploy

SYMPTOM: Only 1-2 emails arrive instead of 3
CAUSE: One recipient email failed (system works as designed)
FIX: This is expected resilience - other emails still delivered
